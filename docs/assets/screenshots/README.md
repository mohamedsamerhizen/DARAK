# DARAK Screenshot Assets

This folder is reserved for real reviewed screenshots from the running DARAK backend.

The PNG files in this folder were captured from the running local Swagger UI after `/swagger/v1/swagger.json` returned HTTP 200.

Current screenshots:

- `swagger-overview.png`
- `auth-endpoints.png`
- `admin-modules.png`
- `resident-endpoints.png`
- `guard-access-endpoints.png`

Failed Swagger pages, screenshots containing secrets, and placeholder screenshots should not be committed.
